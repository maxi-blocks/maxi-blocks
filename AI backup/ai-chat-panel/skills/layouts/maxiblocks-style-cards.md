# Skill: MaxiBlocks Style Cards

## Purpose
Style cards let you re-style entire templates or sections quickly, enabling fast design changes without rebuilding layouts. MaxiBlocks ships with a library of style cards, including light/dark variants, to streamline theming across patterns and templates.【F:README.md†L88-L98】

## Key Controls
- **Global restyling:** Apply a style card to update typography, colors, and spacing across a template or section in one step.【F:README.md†L88-L98】

## Common Patterns
- **Template re-skin:** Apply a new style card to a hero + features + CTA stack to generate a distinct visual theme.
- **Pattern variants:** Duplicate a pattern and apply different style cards for quick A/B options.

## Accessibility Notes
- Verify color contrast after applying a style card.
- Ensure typography remains readable across breakpoints.

## Examples
- Use a light style card for marketing pages and a dark card for landing pages.
- Apply the same style card across multiple templates for brand consistency.

## Gotchas / Troubleshooting
- If colors look off, confirm the style card matches the intended theme set (light/dark).【F:README.md†L88-L98】
