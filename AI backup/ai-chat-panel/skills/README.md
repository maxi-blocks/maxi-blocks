# MaxiBlocks Skills Taxonomy

This directory defines skill guides for MaxiBlocks. Use the block-specific guides when creating patterns, templates, or documentation.

## Core block skills
- `blocks/maxiblocks-container.md` (Container Maxi)
- `blocks/maxiblocks-text.md` (Text Maxi)
- `blocks/maxiblocks-button.md` (Button Maxi)
- `blocks/maxiblocks-image.md` (Image Maxi)
- `blocks/maxiblocks-icon.md` (Icon Maxi)
- `blocks/maxiblocks-divider.md` (Divider Maxi)
- `blocks/maxiblocks-group.md` (Group Maxi)
- `blocks/maxiblocks-slider.md` (Slider Maxi)
- `blocks/maxiblocks-number-counter.md` (Number Counter Maxi)
- `blocks/maxiblocks-search.md` (Search Maxi)
- `blocks/maxiblocks-map.md` (Map Maxi)
- `blocks/maxiblocks-video.md` (Video Maxi)

## Skill guide expectations
Each block guide should include:
- What the block is for
- Key settings/controls
- Markup patterns (if relevant)
- Common layout recipes
- Accessibility notes (e.g., alt text, headings)
