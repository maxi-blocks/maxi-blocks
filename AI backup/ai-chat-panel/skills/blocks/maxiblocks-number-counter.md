# Skill: MaxiBlocks Number Counter Maxi

## Purpose
Number Counter Maxi (`maxi-blocks/number-counter-maxi`) creates number counters for highlighting metrics in layouts.【F:src/blocks/number-counter-maxi/block.json†L1-L17】

## Key Controls
- **Block registration:** `maxi-blocks/number-counter-maxi` with the description "Create a number counter."【F:src/blocks/number-counter-maxi/block.json†L1-L13】

## Common Patterns
- **Stats row:** Use multiple counters in a row to highlight KPIs.
- **Icon + counter:** Pair with Icon Maxi for visual emphasis.

## Accessibility Notes
- Provide text labels alongside numbers to clarify meaning.

## Examples
- "1,200+ customers" counter in a stats section.
- "98% satisfaction" counter beside a testimonial.

## Gotchas / Troubleshooting
- Ensure the counter value is readable even if animation is disabled.
