# Skill: MaxiBlocks Map Maxi

## Purpose
Map Maxi (`maxi-blocks/map-maxi`) creates a map with marker and description for location-focused layouts.【F:src/blocks/map-maxi/block.json†L1-L17】

## Key Controls
- **Block registration:** `maxi-blocks/map-maxi` with the description "Create a map with marker and description."【F:src/blocks/map-maxi/block.json†L1-L13】

## Common Patterns
- **Contact section:** Pair with address and contact details.
- **Location highlight:** Map with short description in a card.

## Accessibility Notes
- Provide a text alternative (address or directions) near the map.

## Examples
- Contact page map with address block.
- Office location map in a two-column layout.

## Gotchas / Troubleshooting
- If the map appears too short, increase the block height or container spacing.
