# Skill: MaxiBlocks Search Maxi

## Purpose
Search Maxi (`maxi-blocks/search-maxi`) adds a search bar with an icon to MaxiBlocks layouts.【F:src/blocks/search-maxi/block.json†L1-L17】

## Key Controls
- **Block registration:** `maxi-blocks/search-maxi` with the description "Add a search bar with icon."【F:src/blocks/search-maxi/block.json†L1-L13】

## Common Patterns
- **Header search:** Place in a header group alongside navigation.
- **Search section:** Combine with a heading and supporting text.

## Accessibility Notes
- Ensure the input has a descriptive label or accessible name.

## Examples
- Header search input with icon.
- Standalone "Search our site" section.

## Gotchas / Troubleshooting
- If the search field feels cramped, adjust the surrounding container width and spacing.
