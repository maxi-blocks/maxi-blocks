# Skill: MaxiBlocks Group Maxi

## Purpose
Group Maxi (`maxi-blocks/group-maxi`) combines a set of blocks into a single grouped unit for layout and styling consistency.【F:src/blocks/group-maxi/block.json†L1-L17】

## Key Controls
- **Block registration:** `maxi-blocks/group-maxi` with the description "Combine a set of blocks in a group."【F:src/blocks/group-maxi/block.json†L1-L13】

## Common Patterns
- **Card content:** Wrap a heading, text, and button into one styled unit.
- **Feature stacks:** Group icon + heading + text for repeated elements.

## Accessibility Notes
- Keep grouped content in logical reading order.

## Examples
- Grouped testimonial card with image, name, and quote.
- Grouped CTA with heading + button.

## Gotchas / Troubleshooting
- Avoid unnecessary nested groups that complicate editing.
