# Skill: MaxiBlocks Slider Maxi

## Purpose
Slider Maxi (`maxi-blocks/slider-maxi`) creates a slider with blocks, controls, and animations for interactive layouts.【F:src/blocks/slider-maxi/block.json†L1-L17】

## Key Controls
- **Block registration:** `maxi-blocks/slider-maxi` with the description "Create slider with blocks, controls and animations."【F:src/blocks/slider-maxi/block.json†L1-L13】

## Common Patterns
- **Testimonial carousel:** Rotate testimonial groups with navigation controls.
- **Hero slider:** Rotate multiple hero variants.

## Accessibility Notes
- Ensure slider controls are keyboard accessible and labeled.
- Avoid auto-advancing without pause controls.

## Examples
- Slider with three testimonial cards.
- Hero slider with image + CTA per slide.

## Gotchas / Troubleshooting
- If slides don't appear, confirm slide blocks are present inside the slider structure.
