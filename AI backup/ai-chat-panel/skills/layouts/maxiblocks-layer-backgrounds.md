# Skill: MaxiBlocks Layered Backgrounds

## Purpose
MaxiBlocks supports layered backgrounds and shape masks to create visually rich sections, including image overlays and composited backgrounds for hero and feature sections.【F:README.md†L156-L159】

## Key Controls
- **Layered backgrounds:** Combine images, colors, and overlays to create depth in sections.【F:README.md†L156-L157】
- **Shape masks:** Use shape masks to add stylistic cut-outs and visual interest to images and sections.【F:README.md†L158-L159】

## Common Patterns
- **Hero overlay:** Background image with a semi-transparent color overlay for readable text.
- **Split background:** Layered color blocks to differentiate columns or callout areas.

## Accessibility Notes
- Ensure sufficient contrast between text and background layers.
- Avoid overly busy backgrounds behind long text blocks.

## Examples
- Hero with background image + dark overlay and light text.
- Feature section with gradient background layer.

## Gotchas / Troubleshooting
- If text readability suffers, increase overlay opacity or adjust text color for contrast.
