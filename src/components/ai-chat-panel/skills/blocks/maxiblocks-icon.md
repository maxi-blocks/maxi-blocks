# Skill: MaxiBlocks Icon Maxi

## Purpose
Icon Maxi (`maxi-blocks/svg-icon-maxi`) adds and styles icons or shapes in MaxiBlocks layouts.【F:src/blocks/svg-icon-maxi/block.json†L1-L17】

## Key Controls
- **Block registration:** `maxi-blocks/svg-icon-maxi` with the description "Add icon or shape and style it."【F:src/blocks/svg-icon-maxi/block.json†L1-L13】

## Common Patterns
- **Feature list:** Icon + heading + description stacks.
- **Accent icons:** Decorative icons next to headings or buttons.

## Accessibility Notes
- Provide accessible labels if icons convey meaning.
- Mark decorative icons so they are ignored by assistive tech.

## Examples
- Service list with one icon per item.
- Icon next to a call-to-action button.

## Gotchas / Troubleshooting
- If icon colors look off, check applied style card or color settings.
