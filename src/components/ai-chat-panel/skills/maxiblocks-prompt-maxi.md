# Skill: MaxiBlocks Prompt Maxi

## Purpose
Prompt Maxi enables AI-assisted editing in MaxiBlocks, offering controls for generating and refining content directly in the editor.【F:README.md†L120-L127】

## Key Controls
- **Generate tab:** Configure content type, tone, writing style, language, temperature, and contextual inputs.【F:README.md†L120-L123】
- **Results tab:** Export, import, clean, replace, generate, and modify AI output.【F:README.md†L125-L125】
- **Modify tab:** Rephrase, shorten, lengthen, fix spelling/grammar, translate, or apply custom edits.【F:README.md†L127-L127】

## Common Patterns
- **Draft copy:** Generate first-pass content for hero headings and summaries.
- **Refine tone:** Use Modify to align copy with brand voice.

## Accessibility Notes
- Review AI-generated text for clarity and readability.
- Ensure headings and link text remain descriptive.

## Examples
- Generate a hero heading with a friendly tone and concise style.
- Use Modify to shorten a long paragraph for mobile layouts.

## Gotchas / Troubleshooting
- If output feels off, adjust tone, writing style, or temperature settings before regenerating.【F:README.md†L120-L123】
