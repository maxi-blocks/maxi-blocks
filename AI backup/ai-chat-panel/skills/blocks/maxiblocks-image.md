# Skill: MaxiBlocks Image Maxi

## Purpose
Image Maxi (`maxi-blocks/image-maxi`) inserts and styles images in MaxiBlocks layouts.【F:src/blocks/image-maxi/block.json†L1-L17】

## Key Controls
- **Block registration:** `maxi-blocks/image-maxi` with the description "Insert, modify or style an image."【F:src/blocks/image-maxi/block.json†L1-L13】

## Common Patterns
- **Hero imagery:** Large, full-width images within a Container Maxi.
- **Card media:** Thumbnail images in grids or feature cards.

## Accessibility Notes
- Provide meaningful alt text for informative images.
- Use empty alt text for purely decorative images.

## Examples
- Product image with caption text below.
- Side-by-side image and text layout.

## Gotchas / Troubleshooting
- If image sizing feels off, review container sizing and image styling controls.
