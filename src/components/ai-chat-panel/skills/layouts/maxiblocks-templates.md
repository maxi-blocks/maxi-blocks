# Skill: MaxiBlocks Templates

## Purpose
Templates are full-page or multi-section compositions built from MaxiBlocks patterns and blocks. MaxiBlocks ships with a large template library to accelerate page creation and standardize layouts across sites.【F:README.md†L35-L214】

## Key Controls
- **Template library usage:** Templates and patterns are accessible from the editor via the Cloud library Maxi block for quick insertion and editing.【F:README.md†L337-L345】【F:src/blocks/cloud-maxi/index.js†L21-L33】

## Common Patterns
- **Hero section:** Heading, supporting copy, CTA button.
- **Feature section:** Icon + heading + text stacks.
- **CTA section:** Focused message and button.
- **Footer:** Multi-column links and contact details.

## Accessibility Notes
- Verify heading order across the full template.
- Ensure interactive elements remain keyboard accessible.
- Provide text alternatives for media sections.

## Examples
- Build a landing page by combining a hero, feature grid, and CTA.
- Reuse a footer template across multiple page templates.

## Gotchas / Troubleshooting
- If sections feel inconsistent, apply a style card across the template to align typography and colors.【F:README.md†L88-L98】
