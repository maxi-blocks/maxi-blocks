# Skill: MaxiBlocks Divider Maxi

## Purpose
Divider Maxi (`maxi-blocks/divider-maxi`) creates a divider between visual elements in MaxiBlocks layouts.【F:src/blocks/divider-maxi/block.json†L1-L17】

## Key Controls
- **Block registration:** `maxi-blocks/divider-maxi` with the description "Create a divider between visual elements."【F:src/blocks/divider-maxi/block.json†L1-L13】

## Common Patterns
- **Section separation:** Place between major page sections.
- **Content grouping:** Separate card stacks or list items visually.

## Accessibility Notes
- Don't rely on dividers as the only indicator of hierarchy.

## Examples
- Divider between a hero section and a feature grid.
- Divider between FAQ items.

## Gotchas / Troubleshooting
- If divider spacing feels tight, adjust margin/padding on surrounding blocks.
